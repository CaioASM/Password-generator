# Pass Generator

This program is a Pass Generator that enables users to create secure passwords with various customization options. It provides an intuitive interface where users can specify the password length and include or exclude character sets such as lowercase, uppercase, numbers, and symbols. Additional features include options to exclude duplicate characters and include spaces in the password for added complexity.

## Features
- Adjustable password length for enhanced security.
- Inclusion options for lowercase, uppercase, numbers, and symbols.
- Ability to exclude duplicate characters to ensure character variety.
- Option to include spaces in the password for complexity.
- Instant password generation with a single click.
- A copy feature to easily copy the generated password to the clipboard.

## Usage
1. Adjust the slider to set the desired password length.
2. Check or uncheck the options for including specific character sets.
3. Optionally, enable the exclusion of duplicate characters and the inclusion of spaces.
4. Click the "Generate Password" button to create a new password.
5. Use the copy button to copy the generated password to the clipboard.

This Pass Generator is designed with user security in mind, providing a customizable and straightforward approach to generating strong, secure passwords.
